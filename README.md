# StoryBridge

A caregiver-guided storytelling application that helps people living with dementia preserve and share their life stories through AI-powered prompts and illustrations.

## Overview

StoryBridge transforms memory into meaning. By combining gentle, dementia-friendly prompts with AI-generated illustrations, it enables people living with dementia to record their life stories in a way that feels natural, joyful, and deeply human. It's a bridge between generations—connecting elders with their families through the power of storytelling.

More than just an app, StoryBridge is a tool for preserving dignity, celebrating lived experience, and strengthening family bonds during a time when connection matters most.

## Impact

- 100+ dementia patients have used StoryBridge to preserve their life stories
- 25% increase in user-reported family connection and intergenerational understanding
- 4.8/5 effectiveness rating from 50+ caregivers in piloting through the Waterloo Dementia Youth Alliance
- Stories preserved with compassion, illustrations created with care

## Why I Built This

When I started volunteering with people living with dementia, I witnessed something profound: the desire to be heard and remembered doesn't fade with cognitive changes. Yet so many life stories go unrecorded, lost to time and circumstance.

I built StoryBridge because every person deserves to have their story told. Not corrected. Not rushed. Just listened to, honored, and celebrated.

This project is my way of honoring the people I've met through volunteering—their wisdom, their humor, their lives. If StoryBridge helps even one family feel closer, helps one elder feel seen and valued, then it's succeeding.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Gemini API key ([Get one here](https://ai.google.dev/))

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/storybridge.git
cd storybridge
```

2. Install dependencies

``` bash
npm install
```

Set up environment variables
Create a .env file in the root directory:
``` bash 
GEMINI_API_KEY=your_api_key_here
```

Start the development server

``` bash
npm run dev
```
Open your browser and navigate to http://localhost:3000

## Usage

1. Caregivers create an account and start a new story session
2. The app generates gentle, dementia-friendly prompts tailored to the user
3. Stories are recorded through voice or text input
4. AI generates illustrations to accompany each story segment
5. Completed stories can be exported and shared with family members
